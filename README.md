# greyout-page

Public site + update feed for **Greyout** — a macOS menu-bar app that grays out
your screen to reduce color/notification distraction.

- Served via GitHub Pages (custom domain: **greyout.cc**, once registered).
- Hosts the landing page (`index.html`), the Sparkle update feed (`appcast.xml`),
  and release downloads.
- The app's source lives in the **private** `caasols/greyout` repo; only the
  public website + binaries live here.
